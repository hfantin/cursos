## comnados 

- lista
> hd head
> tl tail 
> ++ junta lista
> [0 | lista] adiciona zero no comeco

- carregar modulo 
> iex nome_modulo.exs 

- recarregar modulo
> r(nome_modulo)

- importar arquivo 
> import file nome_arquivo.exs

- compilar
> elixirc arquivo.ex

- executar scripts
> elixir arquivo.exs

- criar projeto com mix 
> mix new nome_do_projeto
