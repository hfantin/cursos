Criação do projeto

	npm i create-react-app
    ./node_modules/.bin/create-react-app cdc-admin
    cd cdc-admin && npm start
    acessar http://localhost:3000 no navegador

Backend 

    cdc-admin:
        Jar: http://bit.ly/jar-api-curso-react .
        Código fonte: https://github.com/alberto-alura/cdcadmin-api.
        instruções: instalar mysql - apt-get install mysql-server
                    se possuir senha, executar: 
                        java -Dspring.datasource.password=sua_senha -jar cdcreact_<VERSAO>.jar

        ou acessar no Heroku:
            https://cdc-react.herokuapp.com/api/autores
